  个人简历

晋军政，男，本科学历，1998年出生于河南周口，
该生在校期间学习、生活、工作表现突出，大一的学年里他设定好自己的学习计划，按着自己的目标迈进，努力扎实的学习专业知识，大学的学习让他学会了很多，大学期间无挂科记录，成绩也是保持着班级前三名，大二上学期他所有课程成绩曾在年级专业上排名第一名，他课外到图书馆涉猎，扩展自己的视野，增长了知识，自我学习能力强。并多次获得学校“风华学子奖学金”二等奖，“风华学子奖学金”三等奖，“学习进步奖”,“2020届优秀毕业生”等荣誉称号。生活上勤俭节约，在保证学习的前提下参加社会实践活动，他做过勤工助学和家教等。提高了自己的社会经验个动手能力。他待人热情诚恳，乐于助人，帮助解决大家遇到的问题。并帮助许多同学解决在学习上的问题，担任寝室长，每天打扫宿舍，营造良好的氛围。他在思想上我积极要求进步，树立了良好的人生观和道德观；永远保持与时俱进。热爱祖国，自觉拥护党和国家的路线、方针、政策，遵纪守法。

主要获得奖项：“风华学子奖学金二等奖”、“风华学子奖学金三等奖”、“学习进步奖”、“2020届优秀毕业生”等，其中，校级奖项4项。

主要项目：python飞机大战项目，仿网易云音乐网站，PHP留言本项目，cocos橡皮怪游戏，unity3D小游戏，使用Linux系统开发电子相册，豆瓣影评微信小程序项目，二次开发十二药网网站项目

学过计算机组成原理，操作系统，数据结构，计算机思维导论，汇编语言等计算机专业课程，学过 Java C C++ python Android PHP HTML5 Javascript 等主流编程语言。

具备软件开发等相关实践经验，有较强的领导力，组织能力和团队精神，多次担任项目组组长带领同学完成项目。做事有条理，责任感强，具备较强的抗压能力;
